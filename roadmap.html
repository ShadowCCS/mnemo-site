Roadmap

Track our progress as we build the ultimate learning platform. New features every 1-4 weeks, completely transparent development.

2025 Q3

Alpha 0.1

The foundation of Mnemo, a fully modular arhitecture that allows for rapid development and easy integration of new features.


  • Modular Architecture: The core of Mnemo, allowing for easy updates and new features.
  • Basic Features: Initial set of features including theme and language swithcing, learning paths, and more.
  • Platform: The first alpha release(s) will only be released on Windows
2025 Q3

Alpha 0.2 - Flashcards

Create, review, and organize flashcards with spaced repetition to help you retain knowledge more effectively.


  • Modes: Includes 3 of the most used flashcard algorithms proven by applications such as Anki and Quizlet.
  • Organize: Organize flashcards into decks and folders.
  • Review: Easy and fast to edit and create. Features such as space to create next to decrease time used.
2025 Q3

Alpha 0.x

Each 1-4 weeks adds a new feature including:


  • Notes: Create, edit, and organize notes with rich text formatting.
  • Games: Made using the same Game Engine as Stardew Valley.
  • Explore: A place for developers to share games and extensions to users.
  • Mindmaps: Sort your thoughts and knowledge in one place.
2025 Q4

Version 1.0

Polished version based on feedback and issues found during alpha versions. Includes full cross-platform support and more.

2025 Q4 - 2026 Q1

Introducing New Features

As mnemo has released, we will be introducing new features and improvements to the application.


Some things we want to achieve:


  • Read Aloud: A new mode, which is also built into the notes tab.
  • Audio Review: Upload material and get a podcast of it.
  • Video Lesson: A custom-made AI algorithm creates a presentation based video on your reference material.
  • Bug fixes, optimizations, and general improvements.
