Interface Overview

Last updated: July 17, 2025 5 min read

The interface is made to be simple and easy to use with more advanced features hidden around the corner. Lets take a look.

Sidebar

The sidebar contains all the main navigation options within the application. It is collapsable and expandable by clicking the hamburger menu icon in the top-left corner.

Dashboard

The dashboard is the opening page of the application. It shows you your statistics and the option to jump back into a study session.

The dashboard is split into two sections;

  • The top section shows your statistics through widgets. You can add or remove widgets by clicking the icon in the top-right corner on the widget section then
  • The bottom section allows you to jump back into a session by double clicking on the session you want to jump back into. If you right click you get more options such as Go To, Export and Delete.

Learning Paths

Learning Paths is a great way to study for exams or tests. You can upload any reference materials you want to study from; including PDFs, text files, youtube videos, and more, or simply write a prompt and let the AI generate a learning path for you.

When the learning path is created, the first thing you will see is the unit overview. Most of them is grayed out, this is because the units are generated as you go. However, if you want to generate them all at once you can press the Generate All button, this allows you to also use the paths fully offline.

How To Use The Learning Paths

Each unit is divided into two sections: theory and practice. The theory section provides an article explaining the unit's concepts, while the practice options offer various ways to apply that knowledge.

A short overview of the practice options:

  • Quiz - Creates a quiz to test your knowledge based on the unit theory (can be transfered to the quizzes tab)
  • Flashcards - Creates flashcards to help you memorize the unit theory (can be transfered to the flashcards tab)
  • Adaptive - The adaptive tab is a bit different. It is meant to be a more complete and interactive way to learn. It splits up the unity theory into even smaller chunks, then give you a few questions based on what you just read, then repeat the process until you have learned the entire unit.

Mindmaps

Mindmaps are an excellent way to visualize and organize your knowledge, and they're incredibly easy to use. To create a new mindmap, just click the Create button in the top-left corner of the mindmaps tab.

When you start a new mindmap, you'll see a single node. Double-click it to edit its content. A single click selects the node, making a blue "+" button appear. Click and drag this button to automatically create a new connected node when you release the mouse. You can also drag this button onto an existing node to connect them.

At the bottom of your screen, you'll find the dock. This is where you can create new nodes, customize styling, and access other features:

  • Edit - Modify node content, change colors, and more.
  • Insert - A menu for adding various node types.
  • Icon - Access an icon library to spice up your mindmap.
  • Format - Apply styling options and control formatting.
  • Node - Adjust node-specific features like type and color.

Commands

Commands are an efficiency tool that lets you quickly change node types without needing the insert menu. To activate a command, simply type a "/" inside a node, then follow it with the desired command.

Available commands include:

  • h1 - Switches to a header node
  • h2 - Switches to a sub-header node
  • card - Creates a card node (contains a title and text)
  • image - Creates an image node
  • list - Creates a list node
  • table - Creates a table node
  • code - Creates a code block node
  • quote - Creates a quote node
  • video - Creates a video node (upload a mp4, or a youtube link)

Explore

Discover a world of new content through the Explore tab. Here, you'll find games, extensions, and more, all designed to enhance your application experience. Mnemo proudly features free, community content, with every item undergoing a thorough review to ensure safety and quality that internet sharing often lacks.

How To Use The Explore

Under the search bar is a navigation bar that allows you to navigate through the different categories of content. The "Explore" tab shows the most popular content, while each tab category shows a complete list.

To get more details about a specific item, you can click the Details button. This will open a new window with more information about the item.

Your downloaded content can be easily found in the Library tab. Publishing content is not yet available, for more information read the Publishing section.

Explore

Library

This is your hub for all downloaded content. Similar to the Explore tab, it features a navigation bar beneath the search bar to help you categorize and find what you're looking for.

To preview themes, simply click the Preview button. For other items, select the Details button to get more information.

You can easily uninstall any content by going into its details and clicking either the Uninstall or Connected button.

Settings

The settings page is where you are able to change things about the application. For instance, you can change the theme, language, and all the other settings you might want to change.

The settings menu is seperated into 5 different sections which you can navigate through with the tabs and the secondary sidebar. These sections are:

  • Apperance - Change the theme, font, and other apperance settings.
  • Preferences - Change the preferences for the application.
  • Data & Storage - Manage your data and storage.
  • About & Support - Information about the application and support.
  • Experimental - Experimental features that are not ready for production.
