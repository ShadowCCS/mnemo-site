Games

Last updated: July 28, 2025 10 min read

This article will cover the basics of how games work in mnemo, and how you could use them to learn.

Many studies have shown that games can be a great way to learn. Mnemo has a built-in game system where developers can create games and publish them. These games use data from the mnemo application through the Mnemo API.

Pro Tip: If you want to create your own game jump to the Games section.

How Do Games Work?

Games for mnemo are built around your learning content. Each game might use different sources of data, but they all work in the same way. In both the "Explore" and "Library" tabs, you can see the data that a game uses by a tag up in the right corner on the item.

That tag will show you what data the game is based around. For instance, if it says "flashcard" you know that you need a flashcard deck to play that specific game.

If it says "mixed" you need to click the "details" button to see what data is used.

How To Get Games?

The first step is to install some games. This is done through the "Explore" tab in the sidebar -> find a game you like -> click the Install button.

You will find your installed games in the "Library" tab in the sidebar with every other downloaded content.

Got your own games? No problem! You can also import games you've created or downloaded from elsewhere. Just hit the Import button in the top-right corner of the "Library" tab.

Be careful when importing from the internet, as it might contain malicious code or not work as expected. Items found in the Explore tab are always verified by us.

The final step is to play the game. This is done by clicking the Play button, which you can find on the game item you just installed.

FAQ

  • Can games be used offline? - most games can be used offline, but some might require an internet connection to work. This depends on the game itself.
  • Can I create my own games? - yes, you can create your own games, visit the "Making Games" documentation here. (Requires some programming knowledge)
  • How do i know the games are safe? - All items found in the Explore tab goes through a verification process to ensure they are safe to use.
